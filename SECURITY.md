# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in MiniDB, please report it
responsibly by emailing the maintainer directly or opening a
[private security advisory](https://github.com/sebtardif/MiniDB/security/advisories/new)
on GitHub.

Please do **not** open a public issue for security vulnerabilities.

You can expect an initial response within 48 hours. Once confirmed, a
fix will be released as soon as practical.
